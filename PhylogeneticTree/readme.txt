Instructions:
	1) Make sure "run.bat" and "PhylogeneticTree.jar" files are in the same location. [e.g. Both files on Desktop]
	2) Run the "run.bat" file
	3) Follow instructions on the command prompt [e.g. Input scoring for match, mismatch, and gap. Input sequences (copy and paste).]
	
Format for sequence input:
	>Organism A:
	ATGCGTACGCTAGTCAGCTAGCTAGTAGCTGTCGTCGTAGTAGCTGATCGATGCTAGCTAGCTAGCTAGTCGATCGTAGCTGCTCGCGCCTACGTAGTAA

	>Organism B:
	ATGCGTACGCTAGTGAGCTAGCTAGTAGCTCTCGTCGTAGTAGCTGATCGTTGCTAGCTAGCTAGCTAGTCGTTCGTAGCTGCTGGCGCCTACGTAGTAA

	>Organism C:
	ATGCCTACGCTACTGTGCTAGCTAGTAGCTCTCGTCGTAGTAGCTGATCGTTGCTAGCTAGCTAGCTAGTCGTTCGTAGCTGCTGACGCCTAAGTAGTAA

	>Organism D:
	ATGCCTACGCTACTGTGCTAGCTAGTCGCTCACGTCGTAGTAGCTGATCGTAGCTAGCTAGCTAGCTAGTCCTTCGTAGCTGATGACGCCTAAGTAGTAA

	>Organism E:
	ATGCCTAGGCTACTGTGCTACCTAGTCGCTCACGTCGTAATAGCTGATCGTAGCTAGCTAGCTCGCTAGTCCTGCGTAGCTGATGACGCCTAAGTAGTAA

	stop

Note***: 
	- Must include keyword "stop" at the end of the sequence with an indent right before the last sequence.
	- Indents in between organisms and their sequences are REQUIRED for the program to read them correctly.
	- Organism name and its sequence must be on separate lines.
